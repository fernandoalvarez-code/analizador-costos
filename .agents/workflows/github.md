# Sincronización con GitHub

Este flujo de trabajo se encargará de trackear todos tus archivos modificados localmente, empaquetarlos en un commit y empujarlos automáticamente a tu rama remota.

1. Sincronizar Cambios al Repositorio Remoto
// turbo
```bash
git add .
git commit -m "Actualizaciones realizadas"
git push
```
