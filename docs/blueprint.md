# **App Name**: Analizador de Costos de Corte

## Core Features:

- Diagnóstico Rápido: Calcula rápidamente el punto de equilibrio entre las herramientas de corte actuales y las propuestas, destacando los posibles ahorros en tiempo de ciclo o rendimiento para justificar las diferencias de precio.
- Informe Detallado: Genera una comparación exhaustiva entre las herramientas de corte actuales y las propuestas (A vs. B), incluyendo CPP, ahorros netos, análisis de ROI y horas de máquina liberadas.
- Gestión de Casos: Permite a los usuarios guardar, cargar, eliminar y buscar casos de éxito directamente dentro de la aplicación, asegurando un fácil acceso a análisis y resultados anteriores para futuras referencias.
- Generación de Informes en PDF: La aplicación debe enviar un informe en formato JSON al backend y el backend debe generar el informe final en PDF en formato A4 y permitir su descarga.
- Perspectivas de Ahorro: Herramienta de IA que utiliza el historial de casos y los parámetros de entrada para proporcionar sugerencias personalizadas sobre posibles cambios de herramientas para aumentar los ahorros del cliente.
- Autenticación de Usuario: Cuentas de usuario seguras con funcionalidad de inicio de sesión (correo electrónico/contraseña) para administrar y asociar los datos creados (clientes, informes, casos) con el ID de usuario respectivo.
- Adaptabilidad Móvil: La aplicación es totalmente responsiva y accesible en dispositivos móviles, asegurando una experiencia de usuario óptima tanto en Android como en iOS, para usar el Diagnostico Rapido.

## Style Guidelines:

- Color primario: Azul fuerte (#2962FF) para transmitir confianza, experiencia y fiabilidad, transmitiendo destreza técnica y precisión.
- Color de fondo: Azul muy claro (#E6F0FF), que proporciona un telón de fondo limpio y profesional que mejora la legibilidad.
- Color de acento: Morado (#A23CF0) para llamar la atención sobre métricas importantes, gráficos y llamadas a la acción, señalando la innovación.
- Fuente del encabezado: 'Space Grotesk' sans-serif para los encabezados; Fuente del cuerpo: 'Inter' sans-serif para el cuerpo. Combine el ambiente técnico de 'Space Grotesk' para los encabezados con la claridad neutra de 'Inter' para una lectura cómoda.
- Iconos nítidos y profesionales relacionados con las herramientas, la eficiencia y el análisis de datos, utilizados en toda la aplicación para representar visualmente las funciones y los puntos de datos.
- Un sistema de navegación claro basado en pestañas permite a los usuarios realizar una transición perfecta entre las secciones de diagnóstico rápido, informe detallado y casos guardados.
- Las transiciones suaves y las animaciones sutiles proporcionan retroalimentación a las interacciones del usuario y mejoran la usabilidad de la aplicación.
